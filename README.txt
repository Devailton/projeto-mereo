#Projeto Mereo
Esse é um Front-End Test da Mereo desenvolvido por Devailton Alex da Silveira;

#Configurando ambiente - Ubuntu 16.04
	- Instale o Noje.js: sudo apt-get install nodejs
	- Verifique se o mesmo foi instalado: node --version
	- Verifique se o npm esta instalado: npm --version
	- Se sim execute: sudo npm install -g @angular/cli
	- BREAK - Verifique se sua rede tenha proxy, caso tenha, configure proxy para npm: sudo npm config set proxy "SEU PROXY"

#Executando aplicação
	- Entre na pasta e execute o comando: npm start
	- Abra o navegador e digite: http://localhost:4200/
	- Pronto! A aplicação estará rodando!
